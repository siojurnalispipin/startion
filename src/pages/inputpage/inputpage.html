Komponen Input

Input sangat penting untuk mengumpulkan dan menangani masukkan data oleh pengguna aplikasi. Di Ionic, input menggunakan komponen ion-input. ion-input hanya digunakan untuk masukkan jenis data teks saja, seperti text, email, tel, password, number, search, dan url. Ionic sebenarnya masih menggunakan elemen dasar HTML untuk input, yaitu input type=”text”, tetapi sudah dikembangkan dan disesuaikan dengan desain antarmuka aplikasi mobil.
