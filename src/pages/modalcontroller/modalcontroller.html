MODAL CONTROLLER

Modal adalah panel konten yang berjalan diatas halaman aplikasi yang sedang aktif. Biasanya digunakan untuk membuat pilihan, atau operasi edit data. Modal di Ionic menggunakan NavController untuk menampilkan konten di dalam stack root nav. Modal ditambahkan ke stack seperti cara kerja NavController.push. Ketika sebuah modal disajikan ke controller nav, overlay ditambahkan ke nav root aplikasi. Setelah modal disajikan, kemudian modal dapat ditutup dari dalam komponen modal itu sendiri dengan menggunakan ViewController dismiss. Untuk menyajikan Modal dengan ModalController, gunakan method ModalController.create(), dan present untuk menampilkan Modal ke dalam layar aplikasi

Passing data dan event dismiss

Modal juga dapat mengeluarkan data, dan menerima data yang nantinya dapat berguna ketika digunakan untuk proses tambah atau data. Misalnya kita membuat halaman modal dengan input, dan button, saat button diklik modal ditutup dan mengirimkan data ke halaman yang sedang aktif di stack root nav. Untuk membuat event dan mengecek apakah Modal mengirimkan data, dapat digunakan event onDidDismiss setelah create modal.
