Action Sheet

Action Sheet merupakan dialog yang memungkinkan pengguna memilih action dari serangkaian pilihan. Sebuah Action Sheet dibuat dari serangkaian array buttons dengan setiap button memiliki property seperti text, role, icon, dan handler.
Untuk menampilkan Action Sheet di aplikasi mobil, Ionic menyediakan class library ActionSheetController. Property role pada button terdiri dari cancel, dan destructive. Button tanpa property role akan ditampilkan dengan tampilan default sesuai template platform. Button dengan role cancel akan selalu diletakkan pada posisi bawah button Action Sheet.
Button dengan role destructive hanya akan berlaku efektif pada mode ios. ActionSheetController termasuk dalam library ionic-angular, untuk dapat menggunakannya harus di import ke dalam controller
