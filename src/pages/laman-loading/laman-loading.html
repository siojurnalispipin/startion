Loading

Komponen Loading kadang dibutuhkan oleh aplikasi sebagai feedback ke pengguna untuk mengindikasikan bahwa aplikasi sedang aktif memproses/berpikir/menunggu. Di Ionic terdapat dua komponen atau service yang sering digunakan sebagai indikator loading, yaitu ion-spinner dan LoadingController.


Spinner Loading


Loading Controller

Selain menggunakan ion-spinner, Ionic juga menyediakan class library untuk menampilkan indikator loading. Berbeda dengan ion-spinner yang bisa langsung dipanggil pada view (HTML), LoadingController digunakan melalui controller (.*ts).
Overlay dari LoadingController saat tampil akan menutup keseluruhan dari layar aplikasi, sehingga memblokir interaksi pengguna dengan aplikasi. Method create, dan present untuk membuat atau menampilkan loading. Method dismiss untuk menutup loading.
